Términos de Servicio

1. Aceptación de los Términos

Al acceder y usar PoseidonUI (el "Bot"), aceptas estar sujeto a estos Términos de Servicio y cumplir con todas las normas y políticas aplicables de Discord. Si no estás de acuerdo con alguna parte de estos términos, no uses el Bot.

2. Descripción del Servicio

PoseidonUI es un bot multifunción para Discord que ofrece servicios como:

  • Moderación automática y manual
  • Música y reproducción de audio
  • Mascotas RPG y economía
  • Notificaciones de streaming
  • Configuración personalizada por servidor
  • Panel de administración web

3. Uso Aceptable

Al usar PoseidonUI te comprometes a no:

  • Usar el Bot para actividades ilegales o no autorizadas
  • Intentar acceder o manipular el código del Bot sin permiso
  • Abusar de los comandos para causar spam o distraer
  • Violentar los Términos de Servicio de Discord
  • Usar el Bot para acoso, discriminación o maltrato
  • Intentar sobrecargar el servicio con peticiones excesivas

4. Planes y Suscripciones

PoseidonUI ofrece varios niveles de servicio:

  • Básico: Acceso a funciones básicas
  • Pro: Funciones adicionales (Economía, Niveles, etc.)
  • Élite: Acceso completo (Mascotas v2, Economía Avanzada, etc.)
  • Custom: Personalización completa y marca blanca

Las suscripciones se gestionan manualmente. No se realizarán reembolsos parciales para periodos de suscripción sin usar.

5. Privacidad y Datos

PoseidonUI puede almacenar datos como:

  • IDs de servidor, usuarios y canales
  • Configuraciones personalizadas
  • Información de mascotas, economía y actividad del servidor
  • Logs temporales de mensajes para moderación

No compartimos tus datos con terceros, a excepción de lo necesario para operar el Bot y cumplir con las leyes aplicables. Para más información, consulta nuestra Política de Privacidad.

6. Propiedad Intelectual

Todo el contenido y código de PoseidonUI es propiedad exclusiva de sus desarrolladores. No puedes copiar, modificar, distribuir o usar el código del Bot sin autorización escrita.

7. Disponibilidad y Modificaciones

Hacemos todo lo posible para que PoseidonUI esté disponible 24/7, pero no podemos garantizar disponibilidad continua. Nos reservamos el derecho de modificar o interrumpir el servicio en cualquier momento sin aviso previo.

8. Limitación de Responsabilidad

PoseidonUI se proporciona "tal cual" sin garantías de ningún tipo, expresa o implícita. No nos hacemos responsables por daños directos, indirectos, incidentales, especiales o consecuentes que resulten del uso o la imposibilidad de usar el Bot.

9. Modificaciones de los Términos

Podemos actualizar estos Términos de Servicio de vez en cuando. Cuando lo hagamos, publicaremos la actualización en esta página y, si es un cambio importante, te notificaremos por medio del servidor de soporte.
