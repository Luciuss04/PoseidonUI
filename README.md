## 🔱 ¿Qué es PoseidonUI?

PoseidonUI es un bot de Discord **multifunción y exclusivo**: no hay enlace público de invitación. Cada instalación se realiza de forma personalizada para garantizar estabilidad, rendimiento óptimo y configuración adaptada a cada servidor.

**¿Por qué exclusivo?**  
La mayoría de bots públicos sufren cortes, saturación y falta de soporte. PoseidonUI prioriza la calidad sobre la cantidad: activaciones controladas, soporte directo y configuración inicial guiada incluida en todos los planes.

## ❓ FAQ

**¿Por qué no hay enlace de invitación público?**  
Para garantizar exclusividad, estabilidad y una configuración óptima para cada servidor. Cada instalación es personalizada.

**¿Hay prueba gratuita?**  
Sí. Ofrecemos una demo de 7 días con funciones clave activas para que compruebes el rendimiento antes de contratar.

**¿Puedo cambiar de plan?**  
Sin problema. Puedes subir o bajar de plan en cualquier momento. El bot se ajusta automáticamente de forma inmediata.

**¿Cuánto tarda la activación?**  
Menos de 24 horas desde que confirmas el plan. En la mayoría de casos, en pocas horas.

**¿Qué necesito para empezar?**  
Solo permisos de administrador en tu servidor. Nosotros nos encargamos de todo lo demás.
